📞 Número principal vigente

Cómo funciona

Cada equipo tiene un inicio de usuario (ej: ato) y un número principal vigente. Cuando un usuario entra (ej: atojoaquin), automáticamente se le muestra el número de ese equipo. Para rotar el número, modificalo acá y guardá.

Se usa como número por defecto. Dejalo vacío si querés que esos usuarios no vean ningún número.
Se muestra arriba a la izquierda en la app del usuario. Dejalo vacío si no querés que esos usuarios vean ningún nombre de equipo.

💬 Links de comunidad

Cómo funciona

Cada equipo tiene un inicio de usuario (ej: ato) y un link de comunidad (link de WhatsApp, Telegram, etc.). Cuando el usuario entra (ej: atojoaquin), se le muestra el link de su equipo en el home.

Dejalo vacío si no querés que esos usuarios vean ningún link.

📥 Importar líneas desde Drive

Cómo funciona

Subí un archivo .xlsx exportado desde Google Sheets (Archivo → Descargar → Microsoft Excel). Cada hoja del archivo representa una línea: el nombre de la hoja debe ser el número de teléfono de esa línea (ej: +5491111…) y dentro debe haber una columna de usernames (la primera fila se asume header y se ignora).

El campo Nombre del equipo abajo se aplica a todas las hojas del archivo (ej: "Atomic" o "Argentum"). Para cada Drive distinto subís un archivo distinto.

Vista previa te muestra qué pasaría sin escribir nada. Después apretás Confirmar y se aplica.

Cargando estadísticas…
En las celdas del .xlsx podés escribir solo el sufijo (ej: joaquin398) — el sistema le concatena el prefijo automáticamente para matchear contra atojoaquin398 en la DB. Si la celda ya tiene el prefijo (atojoaquin398) también funciona.

Zona peligrosa

Si cargaste un Drive equivocado, podés limpiar todas las asignaciones de un equipo. Los usuarios afectados volverán a usar el matcher por prefijo de la sección "Números vigentes".
