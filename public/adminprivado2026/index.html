📞 Número principal vigente

Cómo funciona

Cada equipo tiene un inicio de usuario (ej: ato) y un número principal vigente. Cuando un usuario entra (ej: atojoaquin), automáticamente se le muestra el número de ese equipo. Para rotar el número, modificalo acá y guardá.

Se usa como número por defecto. Dejalo vacío si querés que esos usuarios no vean ningún número.
Se muestra arriba a la izquierda en la app del usuario. Dejalo vacío si no querés que esos usuarios vean ningún nombre de equipo.

📋 Asignar línea por listado de usuarios

Subí un archivo .xlsx con una sola columna de usernames (ej: atojoaquin398). Cada archivo es UNA línea de WhatsApp.

Si el usuario ya está registrado en tu plataforma, se le asigna esta línea ahora mismo (pisa la línea del equipo).
Si todavía no se registró, queda guardado en una lista de espera. Cuando entre por primera vez, ve este número como su línea principal automáticamente.

Si la cargás, el jugador ve "TIGER · Línea 1". Si no, solo el equipo.

💬 Links de comunidad

Cómo funciona

Cada equipo tiene un inicio de usuario (ej: ato) y un link de comunidad (link de WhatsApp, Telegram, etc.). Cuando el usuario entra (ej: atojoaquin), se le muestra el link de su equipo en el home.

Dejalo vacío si no querés que esos usuarios vean ningún link.

➕ Nueva estrategia de recuperación

🚨 Caída de línea

¿Se cayó una línea?

Elegí el equipo afectado, poné el nuevo número y mandá un push solo a esos clientes. Opcionalmente activá una promo (% extra por X días) para incentivar la migración. El número se actualiza automáticamente en User, UserLineLookup y en el slot de "Números vigentes".

Tip: elegí "modo prefijo" si tenés varias sub-líneas y se cayó todo el grupo.
